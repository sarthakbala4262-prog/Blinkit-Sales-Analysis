# 🛒 Blinkit Sales Analysis

End-to-end data analytics project on Blinkit 
(India's quick commerce platform) using Python, 
SQL Server, and Power BI.

## 🔧 Tools Used
- Python (Pandas, NumPy) — ETL & data cleaning
- SQL Server / SSMS — insight finding
- Power BI — dashboard & visualisation

## 📊 Key Insights
- GMV: ₹11.01M | AOV: ₹2.20K | Total Orders: 5000
- Delay Rate: 61.96% — critical operational issue
- Top Market: Orai | Top Product: Pet Treats
- Peak Day: Sunday | Avg Delivery: 4.4 mins

## 📁 Project Structure
- data/raw — original messy CSV files
- data/clean — cleaned and transformed files
- python — ETL pipeline script
- sql — T-SQL insight queries
- powerbi — dashboard (.pbix file)
- screenshots — dashboard preview images

## 🗂️ Dataset
5 tables: Orders, Customers, Products, 
Delivery, Inventory
